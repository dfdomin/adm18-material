# NotebookLM prompt - Semana 5

## Fuente unica
Usa solo este archivo.

## Tema
Semana 5: Parcial 1 - Envio del Pacifico.

## Base de contenido
Esta semana no introduce un tema nuevo: sirve para consolidar lo visto en las semanas 1 a 4. El video debe preparar al estudiante para resolver un caso espejo de evaluacion, identificando documentos, corrigiendo errores y aplicando la logica de la GTC 185.

## Lo que debe quedar claro
- que el parcial evalua clasificacion documental y construccion basica
- como leer un caso y detectar que informacion falta o sobra
- como revisar formato, tono y componentes antes de responder
- como usar estrategias de eliminacion de errores para llegar a la opcion correcta
- conexion con comercio exterior: documentos y respuestas que no pueden quedar ambiguas

## Instruccion para NotebookLM
Actua como tutor audiovisual y coach de repaso para estudiantes de comercio exterior.
Quiero un video de 4 a 6 minutos, en espanol simple, usando solo esta fuente.
No quiero una clase nueva: quiero una guia de preparacion para el parcial.
Primero resume lo mas importante de las semanas 1 a 4.
Despues muestra como pensar frente a un caso de evaluacion.
Al final deja un plan breve de estudio y errores comunes a evitar.

## Ejemplos de analogia que puedes usar
- repasar antes de un examen con una lista de chequeo
- revisar un equipaje antes de viajar
- corregir una carta antes de enviarla
- buscar pistas en un caso para no responder a ciegas

## Entregable
- guion narrado
- lista de "lo que mas preguntan"
- 5 errores comunes
- 5 pistas para resolver el caso
- cierre con 3 preguntas de autoevaluacion

## Tono
Estrategico, calmado y orientado a examen.
